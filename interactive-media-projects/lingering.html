Lingering

See Demo

I acted in this project as a Pixel Artist, Animator, Game Programmer, and Game Designer.

I utilized Aseprite and Photoshop for animations and art assets, and C# in Unity2D as a game engine.

ONE

Product

Lingering is a short narrative game made to explore using Unity to tell a story and creating original art assets to enhance the narrative experience. It is a single-player game where you play as a young girl who struggles to let go of the past, and embarks on a dreamlike journey to escape.

The project was completed independently in a 2 week timeframe.

Key Features: Dialogue branching, scene management, multiple choices, side-scrolling camera, interaction outline, scene fading

TWO

Process

I created most pixel assets beforehand, and integrated them into the game code early on. After implementing movement and scene management, I proceeded to tackle the dialogue system and implement multiple routes based on the player's choices. I also added an outline feature to show what interactions were available to improve user feedback.

Some things I learned from the project are:

  • How to create branching dialogue
  • How to create scene animations

THREE

Outcome

3 separate endings for the project were completed. The game is deployed on itch.io as a WebGL file.
